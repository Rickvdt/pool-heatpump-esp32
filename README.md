# Modbus interface for Pool Heatpump
Pulling information from a Fairland (or rebranded) pool heatpump into home assistant. No cloud needed.


## Background
Aim is to use a https://github.com/Xinyuan-LilyGO/T-CAN485 to control the heatpump instead of the WIFI module. Connect on the PCB or the exterior connector using M10/M12 4 pin connector to carrying 12v, Gnd, RS485 A+B),
